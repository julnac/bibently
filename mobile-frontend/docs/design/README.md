## Uwagi

Te pliki służą jako referencje dla deweloperów i nie są wbudowane w aplikację.
Należą do dokumentacji projektu, a nie do assetów runtime.

## TailwindCSS/NativeWind

Aplikacja wykorzystuje TailwindCSS przez NativeWind z rozbudowaną konfiguracją motywu w [tailwind.config.js](../../tailwind.config.js).

### Kluczowe elementy design system:

- **Kolory**: Zdefiniowane w tailwind.config.js z wsparciem dla dark mode
- **Czcionki**: SF Pro Display (iOS-like typography)
- **Spacing**: Standardowa skala TailwindCSS
- **Border Radius**: Zaokrąglone rogi dla nowoczesnego wyglądu
